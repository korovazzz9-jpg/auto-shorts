# Настройка испанского канала (Datos en 60s)

Код уже мультиязычный: переменная окружения `CHANNEL=es` переключает озвучку,
язык сценариев, CTA, субтитры и плейлисты на испанский. Английский канал
(`CHANNEL` не задан или `en`) продолжает работать без изменений.

Осталось сделать руками один раз:

## 1. Создать испанский YouTube-канал (Brand Account)

Отдельный Google-аккаунт НЕ нужен. Под тем же логином:

1. [youtube.com](https://youtube.com) → аватар → **Switch account** → **Create a channel**
   (или [youtube.com/channel_switcher](https://www.youtube.com/channel_switcher) → Create a channel).
2. Назовите канал (например, **Datos en 60s**). Это создаст отдельный Brand Account
   со своими подписчиками/видео, независимый от 60SecFacts.

## 2. Получить отдельный YouTube refresh token для этого канала

Приложение в Google Cloud и `client_secret.json` — те же, что для английского канала.
Меняется только токен (он привязан к конкретному каналу).

```bash
python src/get_youtube_token.py
```

При входе в браузере Google спросит, **каким каналом** авторизоваться —
выберите новый испанский Brand Account (не 60SecFacts!). Скопируйте
напечатанный `YT_REFRESH_TOKEN`.

## 3. Добавить токен как отдельный GitHub-секрет

```bash
gh secret set YT_REFRESH_TOKEN_ES -b"<полученный токен>"
```

Воркфлоу `daily-es.yml` маппит `YT_REFRESH_TOKEN_ES` в стандартное имя
`YT_REFRESH_TOKEN` и выставляет `CHANNEL=es`, так что код автоматически
работает с испанским каналом. Остальные секреты (`ANTHROPIC_API_KEY`,
`PEXELS_API_KEY` и т.д.) общие — отдельно добавлять не нужно.

## 4. Оформление канала (вручную в Studio)

Аватарка, баннер, описание — как делали для английского канала. Можно
сгенерировать испанские версии тем же способом (Pollinations.ai).

## Готово

После этого `daily-es.yml` будет публиковать 5 испанских видео в день
по своему расписанию (на несколько минут смещённому от английского, чтобы
не пересекаться). Instagram для испанского канала пока отключён
(`post_to_instagram: false` в `config.py`) — включим, когда заведёте
отдельный испанский Instagram-аккаунт.
